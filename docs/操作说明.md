## 基本使用流程

```
开局 → 确认庄家 → 开始打牌
  ↓
有人胡牌 → 点击对应胡牌按钮 → 选择玩家 → 选择番型（可选）→ 确认结算
  ↓
弹出结算面板 → 确认得失 → 选择「连庄」或「换庄」→ 进入下一局
  ↓
重复以上步骤…
```

**核心原则：全程一台手机操作，操作完所有人的分数自动计算好，展示给全场核对。**

## 玩家卡片操作

每张卡片显示该玩家的当前状态：

| 区域 | 说明 |
|------|------|
| **名字** | 点击可修改昵称（最多6字） |
| **庄** 圆圈 | 金色=当前庄家；点击任意玩家的「庄」可手动换庄 |
| **重置** | 单独将该玩家总分归零，不影响其他人 |
| **本局** | 当前这局累计得失（局结束后清零） |
| **总分** | 历史累计总分 |

## 胡牌操作

### 🎴 点炮胡

**场景：** A 打出一张牌，B 用这张牌胡牌。

**操作步骤：**
1. 点击「点炮胡」
2. 选择**胡牌者**（B）
3. 选择**点炮者**（A）
4. 可选：勾选番型、开启大胡加倍
5. 点击「确认结算」

**计分规则：**
```
胡牌者（B）：+得分
点炮者（A）：-得分
其余两人：  不变
四人总和：  = 0 ✓
```

---

### ✨ 自摸胡

**场景：** A 自己摸到牌胡了。

**操作步骤：**
1. 点击「自摸胡」
2. 选择**胡牌者**（A）
3. 可选：勾选番型、开启大胡加倍
4. 点击「确认结算」

**计分规则：**
```
胡牌者（A）：+得分 × 3（收三家）
其余三人：  各 -得分
四人总和：  = 0 ✓
```

---

### 🌸 杠上开花

**场景：** A 开杠后摸牌，摸到了胡牌的那张牌。

**操作步骤：**
1. 点击「杠上开花」
2. 选择**胡牌者**（A，即开杠者）
3. 选择**杠的类型**（明杠 / 补杠 / 暗杠）
4. 可选：勾选番型、开启大胡加倍
5. 点击「确认结算」

**计分规则：**
```
第一步 结算杠分：
  A：+杠分 × 3
  其余三人：各 -杠分

第二步 结算胡牌分：
  A：+胡牌分 × 3
  其余三人：各 -胡牌分

最终 A 得到杠分和胡牌分两笔收益
```

---

### 💥 杠上炮

**场景：** A 开杠后摸牌并打出，B 用这张牌胡牌（A 相当于点炮）。

**操作步骤：**
1. 点击「杠上炮」
2. 选择**胡牌者**（B）
3. 选择**开杠者**（A，即点炮方）
4. 选择**杠的类型**
5. 可选：勾选番型、开启大胡加倍
6. 点击「确认结算」

**计分规则：**
```
第一步 结算杠分（A开杠得益）：
  A：+杠分 × 3
  其余三人：各 -杠分

第二步 结算胡牌分（A相当于点炮）：
  B（胡牌者）：+胡牌分
  A（开杠者）：-胡牌分
  其余两人：   不变
```

---

### ⚡ 抢杠胡

**场景：** A 想要补杠（把已碰的牌升级为杠），B 认为这张牌能让自己胡牌，B 抢先胡了，A 赔付。

**操作步骤：**
1. 点击「抢杠胡」
2. 选择**胡牌者**（B）
3. 选择**被抢者**（A，即试图补杠的人）
4. 可选：勾选番型、开启大胡加倍
5. 点击「确认结算」

**计分规则：**
```
胡牌者（B）：+得分
被抢者（A）：-得分
其余两人：   不变（不参与赔付）
四人总和：   = 0 ✓
```

## 杠牌操作

杠牌**不结束本局**，分数立即结算后继续打牌。

### 明杠

**场景：** A 手里有3张一样的牌，另一人打出第4张，A 开杠。

```
A（杠牌者）：+明杠分 × 3
其余三人：   各 -明杠分
```

### 补杠

**场景：** A 已经碰了牌（3张），又摸到第4张，A 补杠。

```
A（杠牌者）：+补杠分 × 3
其余三人：   各 -补杠分
```

### 暗杠

**场景：** A 手里自己摸齐了4张一样的牌，不需要别人打出，A 暗杠。

```
A（杠牌者）：+暗杠分 × 3
其余三人：   各 -暗杠分
（暗杠分通常比明杠/补杠更高）
```

## 番型系统

番型是**特殊牌型加成**，胡牌时如果符合条件可以额外叠加倍率。

| 番型 | 说明 | 默认倍率 |
|------|------|---------|
| **碰碰胡** | 全部由刻子（三张一样）组成，没有顺子 | ×2 |
| **清一色** | 全部是同一花色（全万/全筒/全条） | ×4 |
| **小七对** | 胡了七个不同的对子 | ×2 |
| **龙七对** | 七对里有一对出现4次（双龙对） | ×4 |
| **天胡** | 庄家发完牌直接胡，没摸过牌 | ×8 |
| **地胡** | 闲家第一次摸牌就胡了 | ×4 |
| **清龙** | 手牌包含一套 1-9 的完整顺子 | ×2 |

**多番叠加：** 多个番型同时符合时，倍率**相乘**（不是相加）。

例：清一色（×4）+ 碰碰胡（×2）= 最终 ×8

---

## 大胡加倍

大胡加倍是**临时手动开关**，本次胡牌得分额外再 ×2。

- 适用于：牌型特别出色、或大家协商的特殊场合
- 与番型**叠加**：番型 ×4，再大胡加倍 = ×8
- 每次操作独立设置，不会影响下一局

## 庄家与连庄

### 庄家标识

- 玩家卡片上**金色圆圈「庄」**表示当前庄家
- 点击任意玩家的「庄」圆圈可手动换庄（连庄数同时清零）

### 庄家倍率

当庄家胡牌或开杠时，得分自动 × 庄家倍率（默认 ×2）。

```
庄家自摸：
  庄家：+底分 × 庄家倍率 × 3
  闲家：各 -底分 × 庄家倍率

闲家自摸（庄家为输方之一）：
  庄家付出与其余闲家相同的金额（不翻倍）
```

### 连庄

庄家胡牌后可以选择**连庄**，继续坐庄并获得连庄加成：

| 连庄次数 | 连庄倍率 |
|---------|---------|
| 第1次 | ×2 |
| 第2次 | ×3 |
| 第N次 | ×(N+1) |

连庄倍率与庄家倍率、番型倍率**全部叠加**。

**结算后的选择：**
- 庄家胡牌 → 弹出「连庄」/「换庄」选择
- 闲家胡牌 → 自动换庄，连庄数清零

## 流局

**场景：** 牌摸完了没有人胡牌，本局作废。

在设置页可以配置流局规则：

| 设置项 | 说明 |
|--------|------|
| **启用流局** | 是否允许宣告流局 |
| **流局扣底分** | 流局时庄家收底分，三家各扣底分 |
| **流局保留庄家** | 流局后庄家不变，继续坐庄 |
| **流局清零连庄** | 流局后连庄计数归零 |

## 手动调分

**场景：** 线下协商需要补充调整分数（兜底功能）。

**操作步骤：**
1. 点击「✎ 手动调分」
2. 选择要调整的玩家
3. 输入分数（正数=加分，负数=减分）
4. 点击「确认调整」

⚠️ **注意：手动调分不可撤销**，请确认分数无误再操作。

## 撤销 / 清空 / 重置

| 操作 | 效果 | 可恢复？ |
|------|------|---------|
| **↩ 撤销** | 撤销上一步操作（可连续撤销） | ✅ 可再撤销 |
| **🗑 清空本局** | 清除当前局所有操作，本局分数归零 | ❌ 不可恢复 |
| **重置总分** | 所有玩家总分归零，从第1局重新开始，历史记录保留 | ❌ 不可恢复 |
| 设置页「**清空全部缓存**」 | 清除所有数据，完全恢复出厂 | ❌ 不可恢复 |

## 完整计分公式

所有胡牌/杠牌的最终得分统一用以下公式计算：

```
最终分 = 底分
       × 操作倍率（点炮/自摸/杠上开花/杠上炮/抢杠胡 各自独立设置）
       × 全局倍数
       × 番型倍率（多个番型相乘）
       × 庄家倍率（胡/杠牌者是庄家时生效）
       × 连庄倍率（庄家连续胡牌时叠加）
       × 大胡加倍（手动开启时 ×2）
```

**账目永远平衡：四人得失之和 = 0，无误差。**

### 计算示例

> 底分1，全局倍数1，庄家倍率×2，第1次连庄（×2），碰碰胡（×2），大胡加倍开启

```
庄家自摸：
  底分(1) × 自摸倍率(1) × 全局(1) × 碰碰胡(2) × 庄家(2) × 连庄(2) × 大胡(2)
  = 1 × 1 × 1 × 2 × 2 × 2 × 2 = 16分（每人付16分）

庄家得：16 × 3 = +48分
三家各：-16分
总和：48 - 48 = 0 ✓
```

## 设置说明

进入「⚙️ 设置」可调整以下参数：

### 基础计分

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 底分 | 每次操作的基础分值 | 1 |
| 全局倍数 | 所有操作统一乘以该倍数 | 1 |
| 分数封顶 | 单次操作最高分（0=不限） | 0 |
| 最低负分 | 玩家总分下限（0=不限） | 0 |

### 胡牌倍率

各种胡牌方式可以单独设置倍率（相对于底分）。

### 杠分设置

明杠、补杠、暗杠的每人支付分值可分别配置。

### 庄家 & 连庄

- 可关闭庄家倍率（关闭后庄闲分数完全一致）
- 可设置连庄倍率上限（防止无限叠加）

### 番型倍率

可以为每种番型单独设置倍率，也可以关闭某种番型（关闭后不会出现在选择列表里）。

### 流局规则

见「流局」章节。

## 常见问题

**Q：为什么分数有时候特别高？**
A：检查是否同时叠加了多个倍率（庄家×连庄×番型×大胡），所有倍率相乘会使分数快速增大。在设置页可以调低各项倍率，或设置分数封顶。

**Q：操作错了怎么办？**
A：点击「↩ 撤销」撤回上一步。手动调分不可撤销，需要再手动调回来。

**Q：中途换人怎么处理？**
A：使用玩家卡片上的「重置」按钮将该玩家分数归零，修改昵称为新玩家名字即可。原来的历史分数会保留但不计入新玩家。

**Q：关闭小程序后分数会丢失吗？**
A：不会。所有数据都保存在本机，重新打开即可恢复。
